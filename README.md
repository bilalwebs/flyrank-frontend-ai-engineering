## Getting Started



### Prerequisites

Before you begin, make sure you have the following installed:

- [Git](https://git-scm.com/downloads) — for cloning and managing the repository
- [Node.js](https://nodejs.org/) (v18 or later) — required for React and Next.js projects
- A code editor — [VS Code](https://code.visualstudio.com/) or [Cursor](https://cursor.com/) is recommended



### Clone the Repository

```bash
git clone https://github.com/bilalwebs/flyrank-frontend-ai-engineering.git
cd flyrank-frontend-ai-engineering
```



### Run a Project

When a new project folder is added, navigate into it and follow its README. For a typical Next.js app:

```bash
cd <project-folder>
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
